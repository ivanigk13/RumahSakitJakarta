### Technology Used
+ Vue.js
+ Vue-Router
+ Axios
+ Webpack

### Development
+ Checkout Code : `git checkout https://github.com/mazipan/RumahSakitJakarta.git`
+ Install dependency : `npm install` or `yarn install`
+ Run in local : `npm run dev`
